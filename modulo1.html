MÓDULO I: INICIANTE


Instalação e Estruturas básicas


HISTÓRICO


A Swift foi criada pela Apple como uma alternativa mais moderna e segura à linguagem Objective-C, que era a principal linguagem usada para desenvolver aplicativos iOS e macOS. Ela foi projetada para ser mais rápida, mais legível e menos propensa a erros, com o objetivo de melhorar a produtividade dos desenvolvedores.



CARACTERÍSTICAS


● SINTAXE: A sintaxe da Swift é mais clara e legível, tornando o código mais fácil de entender e manter.

● SEGURANÇA: A Swift foi projetada com foco em evitar erros comuns de programação, como acessos inválidos à memória e problemas de tipos.

● INTERF. DE TIPOS: A linguagem é capaz de inferir automaticamente os tipos das variáveis e constantes, tornando a declaração de variáveis mais concisa.

● OPCIONAIS: Opcionais são uma parte fundamental da Swift, permitindo tratar valores nulos de maneira mais segura.

● MANIPULAÇÃO DE MEMÓRIA: A Swift utiliza a contagem de referência e a coleta de lixo para gerenciar a memória, facilitando a vida dos desenvolvedores.

● PLAYGROUNDS: Os Playgrounds da Swift oferecem um ambiente interativo para testar código rapidamente e visualizar resultados em tempo real.



ESTRUTUTAS DE CONTROLE


A Swift suporta estruturas de controle familiares, como condicionais (if, else) e loops (for, while).

let idade = 18

if idade >= 18 {
print("Você é maior de idade.")
} else {
print("Você é menor de idade.")
}

for i in 1...5 {
print(i)
}

var contador = 0
while contador < 5 {
print("Contador: \(contador)")
contador += 1
}



VARIÁVEIS E CONSTANTES


Na Swift, você pode declarar variáveis usando `var` e constantes usando `let`. Além disso, a inferência de tipos torna a declaração de variáveis mais simples.

var nome = "João"
let pi = 3.14159

nome = "Maria" // Variável pode ser modificada
// pi = 3.14 // Isso resultará em erro, pois 'pi' é uma constante br



OPERADORES


A Swift suporta uma variedade de operadores para realizar operações aritméticas, lógicas e outras.

let soma = 5 + 3
let subtracao = 10 - 4
let multiplicacao = 2 * 6
let divisao = 8 / 2

let resultado = (soma + subtracao) * 2

let verdadeiro = true
let falso = !verdadeiro



TIPOS DE DADOS


A Swift possui tipos de dados básicos, como Inteiros, Pontos Flutuantes, Booleanos e Strings, além da capacidade de criar tipos personalizados.

let inteiro: Int = 42
let pontoFlutuante: Double = 3.14
let booleano: Bool = true
let texto: String = "Olá, Swift!"

// Tipos personalizados
struct Pessoa {
var nome: String
var idade: Int
}

let pessoa = Pessoa(nome: "Ana", idade: 25)
print("Nome: \(pessoa.nome), Idade: \(pessoa.idade)")



FUNÇÕES


As funções são blocos de código que podem ser reutilizados para executar tarefas específicas.

func saudacao(nome: String) -> String {
return "Olá, \(nome)!"
}

let cumprimento = saudacao(nome: "Carlos")
print(cumprimento)



ARRAYS


Arrays são coleções ordenadas de elementos do mesmo tipo.

var numeros = [1, 2, 3, 4, 5]

numeros.append(6)
numeros.insert(0, at: 0)
numeros.remove(at: 3)

for numero in numeros {
print(numero)
}
